# Python-study

점프 투 파이썬 예제와 연습문제를 정리하는 저장소입니다.  
Jupyter Notebook으로 실습하면서, 각 장별로 폴더를 나누어 관리합니다.

## 폴더 구조

- `01.자료형/` : 숫자형, 문자열, 리스트, 튜플, 딕셔너리, 집합 등 기본 자료형 실습
- `02.조건문/` : if, elif, else 조건문 예제와 응용 문제
- `03.함수/` : 함수 정의, 매개변수, 반환값, 스코프 관련 실습
- `04.예외처리/` : try-except, 예외 종류, 예외 발생시키기 등
- `05.클래스` : 클래스,모듈 등
- `09.끄적이기/` : 아이디어 테스트, 코드 메모, 실험용 노트

## 목표

- Python 기본 문법을 손에 익히기
- 책 예제를 그대로 따라치는 것이 아니라,
  - 스스로 변형해보고,
  - 주석으로 개념을 정리하고,
  - Git/GitHub로 학습 이력을 관리하는 것까지 포함
